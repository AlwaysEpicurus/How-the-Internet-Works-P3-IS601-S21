HTTP & HTTPS Protocols

Concept/Definition

The Hypertext Transfer Protocol or HTTP is used to retrieve data from a server (where the data you want to retrieve lives) to a client (your computer or device). HTTP enables the use of links (hyperlinks).

HTTPS is a special form of HTTP that provides added security through encryption (Transport Layer Security), requiring websites to obtain a certificate.

Why we use it

We use HTTP so we can easily get from one page to an another quickly and easily using links! We use HTTPS encrypts your data so bad actors who try intercept your data through man-in-the-middle (MitM) attacks can’t use it because they wouldn’t be able to read it. HTTPS also has protection against client side cookies, which is helpful if you are using computer that are not your own and you don’t want to leave information about yourselves on those computers.
