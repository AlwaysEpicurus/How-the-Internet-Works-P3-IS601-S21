IP Addresses

Concept Definition

Every device and website that is connect and accessible on the internet (via Internet Protocol has an IP Address) has an address called an Internet Protocol Address or IP Address. Most users are unaware of this address because they use Domain Names, such as Google.com, which service as proxies for IP Addresses. The IP Address is made up of 32 bit or 128 bit numbers.IP addresses are management by Internet Assigned Numbers Authority (IANA) which is a group under ICCAN

Why we use it

We use it because it helps user computers to access other computers and information on other computers helping computers identify what we are trying to access, where it is, and how to access it. IP Address are very much like phone numbers.

Examples

116.193.187.223

96.75.135.120

235.455.235.0
