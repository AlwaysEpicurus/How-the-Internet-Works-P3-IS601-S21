IP Addresses

Concept/Definition

Every device and website that is connect and accessible on the internet (via Internet Protocol has an IP Address) has an address called an Internet Protocol Address or IP Address. Most users are unaware of this address because they use Domain Names, such as Google.com, which service as proxies for IP Addresses.

The IP Address is organized in a hierarchy (early version hiearchy represented as region/country,sub network, and specific device) and is made up of 32 bit or 128 bit numbers.IP addresses are management by Internet Assigned Numbers Authority (IANA) which is a group under ICCAN

The IP in IP Address stands for Internet Protocol, a design philosophy and architecture invented by Vint Cerf adn Bob Kahn. It

Currently there are over 4 billion IP Address.

The most current version of IP Addresses is ipv6. It has 128 bits per address meaning it can produce 340 undecillion unique addresses.

Why we use it

We use it because it helps user computers to access other computers and information on other computers helping computers identify what we are trying to access, where it is, and how to access it. IP Address are very much like phone numbers.
