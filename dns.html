DNS

Concept/Definition

Domain Name System or DNS is the switchboard of the internet. It connects domain names to IP addresses. When you type in orangejulius.com you get to see the refreshing fruit drinks because the DNS system connect orangejulius.com to 75.2.85.171.

There isn't just one DNS server. There are many all organized in a distributed hierarchy which splits up top level domains(.com, .us, .io) by zone

Security Concern

Since the DNS system was created to be an open system it is vulnerable to cyber attacks, such as domain name spoofing.

Why we use it

We use it because we need a system to connect domain names with IP address.
